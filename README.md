# FinancialPortal

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 1.0.0.

# Installation

Node: v6.10.2
npm: v3.10.10

Install angular-cli and yarn...

```sh
$ npm install -g @angular/cli
$ npm install -g yarn
```

In the proyect folder...

```sh
$ yarn install
$ ng serve
```
Navigate to `http://localhost:4200/`.
